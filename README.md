# Baidu Dedupe Backup

多设备去重备份项目 PRD。

本项目用于规划一个面向个人和小团队用户的多设备备份产品：用户选择文件或文件夹后，可默认加密打包并备份到百度网盘，同时对比多设备历史备份项目，跳过重复内容，减少云盘空间浪费。
