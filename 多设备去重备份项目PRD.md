# 多设备去重备份项目 PRD

## 1. 文档信息

| 项目 | 内容 |
| --- | --- |
| 产品名称 | 多设备去重备份 |
| 文档类型 | 产品需求文档 PRD |
| 适用端 | 桌面端优先，可扩展移动端与网页管理端 |
| 目标用户 | 有多台电脑或设备、需要长期备份文件到百度网盘的个人用户、小团队用户 |
| 文档范围 | 注册登录、百度网盘授权、设备绑定与管理、备份任务创建与管理、多设备去重、备份进度与异常恢复 |
| 不包含内容 | 技术架构、数据库设计、接口设计、加密算法、文件识别算法、部署方案 |

## 2. 产品背景

用户通常会在多台设备上保存相同或相似的资料，例如工作文档、照片、项目文件、压缩包、安装包等。直接将每台设备的文件全部备份到云盘，会产生重复备份、空间浪费、任务管理混乱、异常中断后难以继续等问题。

本产品希望为用户提供一个统一的多设备备份工具。用户可以选择文件或文件夹，创建备份任务，将内容默认打包加密后备份到百度网盘。产品会对待备份项目与多个已绑定设备的历史备份项目进行对比，自动识别重复内容，减少重复上传和重复占用云盘空间。

## 3. 产品目标

1. 支持用户通过账号体系管理自己的备份数据、设备与备份任务。
2. 支持用户授权并绑定百度网盘，作为备份文件的存储位置。
3. 支持用户绑定多台设备，并查看每台设备的备份情况。
4. 支持选择文件或文件夹创建备份任务。
5. 支持默认加密备份，同时允许用户在创建任务时选择不加密。
6. 支持多设备去重备份，减少重复备份内容。
7. 支持任务开始、暂停、恢复、异常中断恢复、删除和进度查看。
8. 让非专业用户也能清楚理解“哪些内容会被备份、哪些内容已存在、当前备份到哪里了”。

## 4. 用户角色

### 4.1 普通用户

个人用户，拥有一台或多台设备，需要将文件或文件夹备份到自己的百度网盘。

主要诉求：

- 快速选择文件或文件夹备份。
- 默认安全备份，不想理解复杂设置。
- 能节省百度网盘空间。
- 电脑关机、断网、任务异常后可以继续备份。
- 能知道每台设备备份了什么。

### 4.2 多设备用户

同时使用办公电脑、家用电脑、笔记本、外接硬盘等设备的用户。

主要诉求：

- 多设备统一管理。
- 重复资料只备份一次。
- 能区分不同设备上的备份任务。
- 换设备后仍能查看历史备份记录。

## 6. 核心业务流程

### 6.1 新用户首次使用流程

1. 用户打开产品。
2. 进入注册页面并创建账号。
3. 注册成功后进入登录状态。
4. 系统引导用户授权百度网盘。
5. 用户完成百度网盘授权。
6. 系统提示绑定当前设备。
7. 用户确认设备名称并完成绑定。
8. 进入首页，可创建备份任务。

### 6.2 已注册用户使用流程

1. 用户登录账号。
2. 系统检查百度网盘授权状态。
3. 系统检查当前设备绑定状态。
4. 若均已完成，进入首页。
5. 用户选择文件或文件夹创建备份任务。
6. 系统展示待备份内容与去重结果。
7. 用户确认备份设置。
8. 用户开始备份。
9. 用户可查看进度、暂停、恢复或删除任务。

### 6.3 创建备份任务流程

1. 用户点击“创建备份任务”。
2. 用户选择文件或文件夹。
3. 系统生成待备份项目清单。
4. 用户选择是否加密，默认开启加密。
5. 系统对比多设备已备份项目。
6. 页面展示：
   - 本次选择总项目数。
   - 已在其他设备或当前设备备份过的项目。
   - 本次需要实际备份的项目。
   - 因重复而跳过的项目。
7. 用户确认创建任务。
8. 任务进入任务列表。
9. 用户点击开始备份。

### 6.4 异常中断恢复流程

1. 备份过程中出现断网、应用关闭、设备关机或其他异常。
2. 用户再次打开产品并登录。
3. 系统在任务列表中标记异常中断任务。
4. 用户可点击“继续备份”。
5. 产品从上次中断位置附近恢复，已完成的内容不重复处理。
6. 任务继续推进，直到完成或再次异常。

## 7. 功能需求

## 7.1 用户注册

### 功能描述

用户可以通过注册创建产品账号，用于管理个人设备、百度网盘授权、备份任务与备份记录。

### 主要规则

- 用户需要填写手机号或邮箱、密码、确认密码。
- 注册前需要同意用户协议和隐私政策。
- 手机号或邮箱已注册时，提示用户直接登录或找回密码。
- 密码需满足基本安全要求。
- 注册成功后可自动登录，也可跳转登录页，由产品策略决定。本 PRD 建议注册成功后自动登录，减少首次使用阻力。

### 异常提示

- 账号格式错误。
- 密码不符合要求。
- 两次密码输入不一致。
- 验证码错误或过期。
- 账号已存在。
- 网络异常，请稍后重试。

## 7.2 用户登录

### 功能描述

用户通过账号密码登录产品，登录后可进入个人备份空间。

### 主要规则

- 支持手机号或邮箱登录。
- 支持记住登录状态。
- 支持退出登录。
- 支持忘记密码入口。
- 连续输入错误时，需进行安全验证或短暂限制。

### 异常提示

- 账号不存在。
- 密码错误。
- 账号状态异常。
- 网络异常，请稍后重试。

## 7.3 百度网盘授权

### 功能描述

用户需要授权产品访问自己的百度网盘，用于保存备份文件。未完成授权时，用户不能开始备份任务。

### 主要规则

- 用户首次使用时，首页或引导页提示“授权百度网盘”。
- 用户授权成功后，页面显示授权账号信息和授权状态。
- 授权失效时，系统提示重新授权。
- 用户可以主动解除百度网盘绑定。
- 解除绑定前，需要提示可能影响正在进行或后续备份任务。

### 授权状态

- 未授权。
- 已授权。
- 授权已失效。
- 授权异常。

### 用户可见信息

- 百度网盘账号昵称。
- 授权状态。
- 最近授权时间。
- 重新授权入口。
- 解除绑定入口。

## 7.4 设备绑定

### 功能描述

用户可以将当前设备绑定到账号下，用于区分不同设备的备份任务和历史备份内容。

### 主要规则

- 用户首次在新设备登录后，系统提示绑定当前设备。
- 系统提供默认设备名称，用户可修改。
- 同一账号下可绑定多台设备。
- 当前设备未绑定时，可以查看账号信息，但不能创建备份任务。
- 设备绑定成功后，该设备可以创建和执行备份任务。

### 设备信息

- 设备名称。
- 设备类型。
- 首次绑定时间。
- 最近在线时间。
- 最近备份时间。
- 备份任务数量。
- 备份状态概况。

## 7.5 设备管理

### 功能描述

用户可以查看和管理当前账号下绑定的所有设备。

### 主要功能

- 查看设备列表。
- 查看当前设备标识。
- 修改设备名称。
- 查看设备备份概况。
- 解绑设备。
- 查看设备下的备份任务。

### 设备状态

- 当前在线。
- 最近在线。
- 长时间未使用。
- 已解绑。

### 解绑规则

- 用户可以解绑非当前设备。
- 用户解绑当前设备前，需要二次确认。
- 解绑设备不会自动删除百度网盘中已完成的备份文件。
- 解绑设备后，该设备再次使用时需要重新绑定。
- 与该设备相关的历史备份记录仍可在账号内查看。

## 7.6 创建备份任务

### 功能描述

用户可以选择一个或多个文件、一个或多个文件夹创建备份任务。

### 主要规则

- 支持选择文件。
- 支持选择文件夹。
- 支持同时选择多个项目。
- 创建前展示选择结果。
- 默认开启加密备份。
- 用户可选择关闭加密。
- 用户可填写任务名称。
- 未填写任务名称时，系统自动生成名称。
- 用户可选择备份说明，作为个人备注。

### 任务设置项

| 设置项 | 默认值 | 说明 |
| --- | --- | --- |
| 任务名称 | 自动生成 | 用户可修改 |
| 备份内容 | 用户选择 | 文件或文件夹 |
| 加密备份 | 开启 | 用户可关闭 |
| 去重备份 | 开启 | 本期默认开启，不建议关闭 |
| 任务备注 | 空 | 可选 |

### 创建前校验

- 百度网盘未授权时，不允许创建备份任务。
- 当前设备未绑定时，不允许创建备份任务。
- 未选择文件或文件夹时，不允许继续。
- 所选项目不存在或无权限访问时，提示用户处理。
- 百度网盘空间不足时，提示用户清理空间或调整备份内容。

## 7.7 打包与加密备份

### 功能描述

用户选择的文件或文件夹在备份前会被整理为适合保存到百度网盘的备份包。默认情况下，备份包会被加密，提升用户资料的安全性。

### 主要规则

- 默认开启加密。
- 用户可在创建任务时关闭加密。
- 关闭加密前，需要提示风险。
- 已创建任务的加密设置在任务开始后不可随意变更。
- 加密备份任务完成后，用户可在备份记录中看到“已加密”标识。
- 未加密备份任务完成后，用户可看到“未加密”标识。

### 风险提示文案建议

关闭加密后，备份内容将以未加密形式保存到百度网盘。请确认该任务不包含敏感资料。

## 7.8 多设备去重备份

### 功能描述

系统在备份前会将本次待备份项目与当前账号下多台设备的已备份项目进行对比。若发现相同内容已经备份过，则本次任务可跳过重复项目，减少重复备份。

### 主要规则

- 去重范围为当前用户账号下已绑定或曾绑定设备的历史备份项目。
- 同一设备内已备份过的项目也参与对比。
- 重复项目不会重复占用备份空间。
- 去重结果需要在用户确认任务前展示。
- 用户可查看哪些项目会备份，哪些项目会跳过。
- 用户不需要理解去重原理，只需要看到清晰结果。

### 去重结果分类

| 类型 | 说明 |
| --- | --- |
| 需备份 | 本次需要实际备份的项目 |
| 已备份 | 已在当前账号的其他任务中备份过 |
| 重复跳过 | 本次不再重复备份的项目 |
| 异常待确认 | 当前无法确认状态，需要用户处理或稍后重试 |

### 页面展示要求

- 展示总文件数或总项目数。
- 展示预计需要备份的项目数。
- 展示可跳过的重复项目数。
- 展示预计节省的空间。
- 支持展开查看重复项目详情。
- 支持按设备来源查看重复项来自哪台设备。

## 7.9 备份任务管理

### 功能描述

用户可以在任务列表中查看、操作和管理所有备份任务。

### 支持操作

- 开始任务。
- 暂停任务。
- 恢复任务。
- 异常中断后继续任务。
- 删除任务。
- 查看任务详情。
- 查看备份进度。
- 查看备份结果。

### 任务状态

| 状态 | 说明 |
| --- | --- |
| 待开始 | 任务已创建，但尚未开始 |
| 准备中 | 正在整理备份内容 |
| 对比去重中 | 正在生成去重结果 |
| 备份中 | 正在备份 |
| 已暂停 | 用户主动暂停 |
| 恢复中 | 正在从暂停或异常中断状态恢复 |
| 异常中断 | 因断网、关闭应用、设备关机等原因中断 |
| 备份失败 | 任务无法继续，需要用户处理 |
| 已完成 | 所有需备份项目已完成 |
| 已删除 | 用户已删除任务记录 |

### 删除规则

- 待开始、已暂停、异常中断、失败、完成的任务均可删除。
- 正在备份中的任务需要先暂停或确认后删除。
- 删除任务记录不默认删除百度网盘中已完成的备份文件。
- 删除前需要明确提示删除影响。

### 删除确认文案建议

删除后，该任务将不再显示在任务列表中。已完成上传到百度网盘的备份文件不会自动删除。

## 7.10 备份进度查看

### 功能描述

用户可以在任务列表和任务详情页查看备份进度，了解当前处理到哪一步。

### 进度信息

- 当前任务状态。
- 总项目数。
- 已完成项目数。
- 跳过重复项目数。
- 异常项目数。
- 当前正在处理的项目名称。
- 已用时间。
- 预计剩余时间。
- 当前备份速度。
- 备份目标位置。

### 展示要求

- 任务列表显示简洁进度条。
- 任务详情显示更完整的分项进度。
- 异常信息需要明确说明原因和建议操作。
- 完成后展示备份结果汇总。

## 7.11 异常处理与恢复

### 功能描述

备份任务可能因断网、电脑关机、应用关闭、百度网盘授权失效、空间不足等原因中断。产品需要帮助用户理解问题，并尽可能恢复任务。

### 异常类型

| 异常类型 | 用户提示 | 用户操作 |
| --- | --- | --- |
| 网络异常 | 网络连接异常，备份已暂停 | 检查网络后继续 |
| 应用关闭 | 上次任务未正常完成 | 继续备份 |
| 设备关机 | 任务因设备离线中断 | 重新打开后继续 |
| 授权失效 | 百度网盘授权已失效 | 重新授权 |
| 空间不足 | 百度网盘空间不足 | 清理空间后继续 |
| 文件不可访问 | 部分文件无法读取 | 跳过或重新选择 |
| 用户取消 | 用户已取消任务 | 重新开始或删除 |

### 恢复规则

- 异常中断任务应保留在任务列表中。
- 用户重新进入产品后，应看到明确的恢复入口。
- 已完成项目不应重复备份。
- 无法恢复的任务，应提示原因和下一步建议。
- 恢复前可重新进行必要检查，例如授权状态、设备状态、网盘空间。

## 7.12 备份记录

### 功能描述

用户可以查看历史备份记录，了解自己曾经备份过哪些任务、来自哪台设备、是否加密、是否完成。

### 记录内容

- 任务名称。
- 所属设备。
- 创建时间。
- 完成时间。
- 备份内容概况。
- 是否加密。
- 去重结果。
- 任务状态。
- 百度网盘保存位置。

### 查询筛选

- 按设备筛选。
- 按任务状态筛选。
- 按时间筛选。
- 按是否加密筛选。
- 按关键词搜索任务名称。

## 8. 前台页面功能描述

## 8.1 启动页

### 页面目标

承接产品启动状态，判断用户是否已登录，并进入对应页面。

### 页面内容

- 产品名称或品牌标识。
- 加载状态。
- 异常提示。

### 页面逻辑

- 已登录用户进入首页。
- 未登录用户进入登录页。
- 登录状态异常时，引导重新登录。

## 8.2 注册页

### 页面目标

帮助新用户创建账号。

### 页面元素

- 手机号或邮箱输入框。
- 验证码输入框。
- 密码输入框。
- 确认密码输入框。
- 用户协议与隐私政策勾选框。
- 注册按钮。

## 10. 产品规则

## 10.1 账号规则

- 一个用户账号可绑定多个设备。
- 一个设备同一时间建议只绑定一个用户账号。
- 用户退出登录不影响已完成备份记录。
- 用户重新登录后可继续查看历史任务。

## 10.2 备份规则

- 备份前必须完成百度网盘授权。
- 备份前必须完成当前设备绑定。
- 备份任务默认加密。
- 去重备份默认开启。
- 已完成备份的重复内容不再重复备份。
- 异常中断后，用户可以继续任务。
- 删除任务记录不默认删除云盘中的备份文件。

## 10.3 设备规则

- 当前设备需要在设备列表中明确标记。
- 解绑设备不影响历史备份记录查看。
- 解绑后再次使用需重新绑定。
- 设备名称允许重复，但建议提示用户使用易识别名称。

## 10.4 百度网盘规则

- 未授权不能开始备份。
- 授权失效后不能继续备份，需重新授权。
- 空间不足时任务暂停或失败，用户清理空间后可继续。
- 用户主动解绑百度网盘后，未完成任务不能继续。

## 11. 权限与安全提示

本产品需要向用户清楚说明以下事项：

- 选择文件或文件夹后，产品会读取这些项目用于备份。
- 默认加密可提升备份内容安全性。
- 关闭加密会降低备份内容保护能力。
- 百度网盘授权仅用于备份相关操作。
- 用户可以随时解除百度网盘绑定。
- 删除任务记录不等于删除百度网盘文件。

## 12. 用户体验要求

1. 首次使用流程应尽量短，避免用户在授权、设备绑定和创建任务之间迷路。
2. 创建备份任务时，默认设置应适合大多数用户。
3. 加密和去重应以用户能理解的语言解释，不展示复杂技术概念。
4. 任务进度必须清晰，尤其是大文件夹备份场景。
5. 异常中断不可让用户感到任务丢失，应提供明确恢复入口。
6. 删除、解绑、关闭加密等高影响操作必须二次确认。
7. 多设备去重结果要可解释，让用户知道为什么某些项目被跳过。

## 13. 主要提示文案

### 13.1 百度网盘未授权

当前还没有授权百度网盘。完成授权后，备份文件将保存到你的百度网盘。

### 13.2 当前设备未绑定

当前设备尚未绑定。绑定后，你可以在此设备上创建和执行备份任务。

### 13.3 关闭加密确认

关闭加密后，备份内容将以未加密形式保存。请确认该任务不包含敏感资料。

### 13.4 去重完成

分析完成。本次选择的项目中，有部分内容已在你的其他设备或历史任务中备份过，将自动跳过重复项目。

### 13.5 异常中断恢复

上次备份未正常完成。你可以继续备份，已完成的内容不会重复处理。

### 13.6 删除任务确认

删除后，该任务将不再显示在任务列表中。已完成上传到百度网盘的备份文件不会自动删除。

### 13.7 百度网盘空间不足

百度网盘空间不足，当前任务已暂停。请清理空间后继续备份。

## 16. 待确认问题

以下问题不影响 PRD 初稿评审，但建议在进入原型或详细设计前确认：

1. 产品首期是否只做桌面端，还是同时包含网页管理端。
2. 注册方式优先使用手机号、邮箱，还是两者都支持。

6. 是否需要设置单个账号可绑定设备数量上限。
7. 是否需要支持定时自动备份。
