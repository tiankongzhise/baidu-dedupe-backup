# 测试策略

## 1. 文档目的

本文档定义 Baidu Dedupe Backup 首期开发的测试范围、测试层级、关键用例和验收检查清单。它用于指导研发、测试和产品验收。

## 2. 测试目标

1. 验证用户能完成注册、授权、绑定设备、创建任务、备份、查看结果的主路径。
2. 验证默认加密、关闭加密确认、去重展示和删除克制等核心产品原则。
3. 验证断网、关机、应用关闭、授权失效、空间不足、文件不可访问等异常恢复体验。
4. 验证多设备历史记录和去重来源展示正确。
5. 验证不会提交真实用户数据、授权凭据或敏感日志。

## 3. 测试层级

### 3.1 单元测试

覆盖：

- 任务状态机。
- 错误码转换。
- 去重结果分类。
- 文件清单解析。
- 任务进度计算。
- 高影响操作权限判断。

### 3.2 集成测试

覆盖：

- 客户端与服务端 API。
- 任务创建到去重分析。
- 任务进度上报。
- 任务完成写入备份记录。
- 通知创建和处理。
- 百度网盘适配器 Mock。

### 3.3 端到端测试

覆盖：

- 新用户首次使用流程。
- 已注册用户创建备份任务。
- 多设备去重展示。
- 异常中断后继续备份。
- 删除任务和历史记录保留。

### 3.4 手工体验测试

覆盖：

- 风险提示是否清楚。
- 去重结果是否用户可理解。
- 首页是否给出明确下一步行动。
- 错误提示是否包含下一步建议。

## 4. 核心测试场景

### 4.1 新用户首次使用

前置条件：

- 用户未注册。
- 当前设备未绑定。
- 百度网盘未授权。

步骤：

1. 打开产品。
2. 注册账号。
3. 完成百度网盘授权。
4. 绑定当前设备。
5. 进入首页。

预期：

- 注册成功后自动登录。
- 未授权时不能创建备份任务。
- 授权完成后继续设备绑定。
- 设备绑定完成后首页展示创建备份任务入口。

### 4.2 默认加密备份

前置条件：

- 用户已登录。
- 百度网盘已授权。
- 当前设备已绑定。

步骤：

1. 创建备份任务。
2. 选择文件或文件夹。
3. 查看加密开关默认状态。
4. 完成去重分析并创建任务。
5. 开始并完成任务。

预期：

- 加密默认开启。
- 任务详情和备份记录展示“已加密”。

### 4.3 关闭加密确认

步骤：

1. 创建备份任务。
2. 关闭加密开关。
3. 查看风险弹窗。
4. 取消关闭。
5. 再次关闭并确认。

预期：

- 第一次取消后加密保持开启。
- 确认后加密关闭。
- 风险文案说明未加密保存风险。
- 完成后任务展示“未加密”。

### 4.4 多设备去重

前置条件：

- 账号下设备 A 已完成某文件备份。
- 设备 B 登录同一账号并已绑定。

步骤：

1. 在设备 B 选择与设备 A 已备份内容相同的文件。
2. 发起去重分析。
3. 展开重复项目详情。

预期：

- 去重结果展示重复跳过数量。
- 重复项目来源展示设备 A 和历史任务。
- 用户能看到预计节省空间。
- 相同内容但文件名不同仍应判定重复。
- 相同文件名但内容不同不得判定重复。

### 4.5 异常中断恢复

步骤：

1. 开始一个备份任务。
2. 模拟应用关闭或网络中断。
3. 重新打开产品。
4. 点击继续备份。

预期：

- 任务列表展示异常中断状态。
- 首页或通知中心展示恢复入口。
- 继续前检查授权、设备、网络、空间和文件访问。
- 已完成项目不重复处理。

### 4.6 百度网盘授权失效

步骤：

1. 开始备份任务。
2. 模拟授权失效。
3. 尝试继续任务。

预期：

- 任务暂停或异常中断。
- 用户看到重新授权入口。
- 重新授权后可继续任务。

### 4.7 空间不足

步骤：

1. 创建需要上传较大内容的任务。
2. 模拟百度网盘空间不足。

预期：

- 任务暂停或失败，具体状态根据是否可恢复决定。
- 用户看到清理空间后继续的提示。
- 清理空间并检查通过后可继续。

### 4.8 删除任务

步骤：

1. 创建并完成备份任务。
2. 删除任务记录。

预期：

- 删除前出现二次确认。
- 确认文案说明云盘文件不会自动删除。
- 删除后任务不再显示在任务列表中。
- 历史记录或结果仍能说明已完成备份文件位置。

## 5. 状态机测试矩阵

| 当前状态 | 操作 | 预期 |
| --- | --- | --- |
| pending_start | start | 进入 preparing 或 backing_up 前置阶段 |
| pending_start | delete | 进入 deleted |
| backing_up | pause | 进入 paused |
| backing_up | delete | 要求二次确认 |
| paused | resume | 进入 resuming |
| interrupted | continue_after_interruption | 进入 resuming |
| completed | delete | 进入 deleted，云盘文件不删除 |
| deleted | start | 拒绝操作 |

## 6. Mock 与测试数据要求

- 测试账号必须使用虚构手机号或邮箱。
- 测试文件必须使用样例文件，不使用真实用户资料。
- 百度网盘接口测试应优先使用 Mock 或测试账号。
- 授权令牌、密钥、真实路径不得提交到仓库。
- 测试路径示例可使用 `D:/Sample/ProjectA`、`/Users/demo/Documents/Sample`。

## 7. 去重专项测试

去重专项测试以 `dedupe-strategy.md` 为准，至少覆盖：

- 相同内容、不同文件名，判定重复。
- 相同文件名、不同内容，判定不重复。
- 相同大小、不同内容，判定不重复。
- 相同内容、不同修改时间，判定重复。
- `quickFingerprint` 不包含修改时间，且不能作为最终跳过依据。
- 同一文件夹在不同扫描顺序下生成相同文件夹清单指纹。
- 设备解绑后，历史索引仍参与去重。
- 加密任务和未加密任务写入同一格式的源文件内容指纹索引。
- 文件不可访问、指纹生成失败、来源记录缺失时进入异常待确认。

## 8. 压缩校验与磁盘空间专项测试

压缩、解包校验和临时空间测试以 `dedupe-strategy.md` 为准，至少覆盖：

- 压缩包完整性测试失败时，任务不得进入上传阶段。
- 压缩包可解但解出清单与源清单不一致时，任务不得进入上传阶段。
- 解包校验通过后，解包目录应被清理。
- 本地临时空间不足时，任务应暂停或异常中断，并提示用户清理磁盘后继续。
- 应用异常退出后，下一次启动能识别任务临时工作区并恢复或清理。

## 9. 验收检查清单

- 注册、登录、退出登录可用。
- 未登录用户不能访问受保护页面。
- 未授权不能创建或开始备份。
- 当前设备未绑定不能创建或执行备份。
- 加密默认开启。
- 关闭加密有二次确认。
- 去重分析结果清晰展示。
- 任务可开始、暂停、恢复、继续和删除。
- 异常中断后可恢复。
- 删除任务不默认删除百度网盘文件。
- 解绑设备不影响历史记录查看。
- 授权失效、空间不足、文件不可访问均有明确下一步建议。
