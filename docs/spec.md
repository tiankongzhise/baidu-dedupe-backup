# Baidu Dedupe Backup Spec

## 1. 文档目的

本文档基于 `多设备去重备份项目PRD.md`，将产品需求整理为后续产品设计、原型设计、研发拆分和验收测试可直接使用的功能规格。

本文档回答“首期要做什么、做到什么程度、如何判断完成”，不展开具体技术架构、数据库表结构、接口协议、加密算法或文件指纹算法。

## 2. 产品概述

Baidu Dedupe Backup 是一款桌面端优先的多设备去重备份工具。用户登录后，可授权百度网盘、绑定当前设备，选择文件或文件夹创建备份任务。任务默认加密打包后备份到百度网盘，并在备份前与同一账号下多设备历史备份记录进行对比，跳过重复内容，减少重复上传和云盘空间浪费。

## 3. 产品目标

1. 让用户通过统一账号管理设备、备份任务、备份记录和百度网盘授权。
2. 让用户可以快速选择文件或文件夹，并以默认安全的方式完成备份。
3. 通过多设备历史备份对比，减少重复备份与云盘空间浪费。
4. 在断网、关机、应用关闭、授权失效、空间不足等异常场景下保留任务，并提供明确恢复入口。
5. 用普通用户能理解的语言解释加密、去重、异常恢复和授权状态。

## 4. 目标用户

### 4.1 普通个人用户

- 拥有一台或多台电脑。
- 需要长期把重要文件或文件夹备份到自己的百度网盘。
- 希望默认设置安全、简单，不需要理解复杂技术细节。

### 4.2 多设备用户

- 同时使用办公电脑、家用电脑、笔记本、外接硬盘等设备。
- 希望在同一账号下统一查看每台设备的备份情况。
- 希望重复资料只备份一次，并能知道重复内容来自哪台设备或哪个历史任务。

## 5. 范围定义

### 5.1 首期包含

- 注册、登录、退出登录。
- 忘记密码或重置密码入口。
- 百度网盘授权、重新授权、解除绑定。
- 百度网盘授权状态查看。
- 当前设备绑定。
- 设备列表、设备重命名、设备解绑、设备备份概况。
- 文件或文件夹选择。
- 创建备份任务。
- 创建任务前展示待备份项目清单。
- 默认加密备份，允许用户选择不加密。
- 关闭加密风险确认。
- 多设备去重分析。
- 展示需备份项目、已备份项目、重复跳过项目、异常待确认项目。
- 任务开始、暂停、恢复、异常中断后继续、删除。
- 任务进度、任务详情、备份结果查看。
- 历史备份记录查询。
- 空状态、加载状态、错误状态和高影响操作确认。

### 5.2 首期不包含

- 百度网盘之外的其他云盘。
- 企业管理员后台。
- 团队共享备份空间。
- 文件在线预览与在线编辑。
- 跨账号迁移。
- 自动同步所有系统目录。
- 手机相册自动备份。
- 技术参数配置页面。
- 备份完成后的本地恢复功能。
- 定时自动备份。

## 6. 核心原则

1. **默认安全**：备份任务默认开启加密，关闭加密必须二次确认并提示风险。
2. **用户可理解**：去重、异常恢复、授权状态等说明应面向普通用户，不暴露复杂技术细节。
3. **多设备统一**：设备、任务、备份记录都围绕同一用户账号统一管理。
4. **异常可恢复**：断网、关机、应用关闭等中断不能让用户感觉任务丢失。
5. **删除要克制**：删除任务记录不默认删除百度网盘中已完成的备份文件。
6. **高影响操作需确认**：解绑设备、解绑百度网盘、关闭加密、删除任务都需要二次确认。

## 7. 用户旅程

### 7.1 新用户首次使用

1. 用户打开产品。
2. 用户注册账号。
3. 注册成功后自动登录。
4. 系统引导授权百度网盘。
5. 用户完成授权后，系统展示授权成功状态。
6. 系统引导绑定当前设备。
7. 用户确认或修改设备名称，并完成绑定。
8. 用户进入首页，看到创建备份任务入口。

### 7.2 已注册用户日常使用

1. 用户登录账号。
2. 系统检查百度网盘授权状态。
3. 系统检查当前设备绑定状态。
4. 授权与设备状态正常时，用户进入首页。
5. 用户创建备份任务并选择文件或文件夹。
6. 系统读取项目清单并执行去重分析。
7. 用户查看去重结果并确认创建任务。
8. 用户开始备份。
9. 用户在任务列表或详情页查看进度，并可暂停、恢复或删除任务。

### 7.3 异常中断恢复

1. 备份过程中出现断网、应用关闭、设备关机或其他异常。
2. 系统保留任务状态与已完成进度。
3. 用户再次打开产品并登录。
4. 任务列表突出显示异常中断任务。
5. 用户点击“继续备份”。
6. 系统重新检查授权、设备、网络、空间和文件可访问状态。
7. 检查通过后，任务从上次中断位置附近恢复，已完成内容不重复处理。

## 8. 功能规格

### 8.1 账号注册

**目标**：让新用户创建产品账号，用于管理设备、授权、任务和备份记录。

**输入信息**：

- 手机号或邮箱。
- 验证码。
- 密码。
- 确认密码。
- 用户协议与隐私政策勾选状态。

**规则**：

- 手机号或邮箱格式必须合法。
- 注册前必须同意用户协议和隐私政策。
- 密码必须满足基础安全要求。
- 两次密码必须一致。
- 已注册账号应提示用户登录或找回密码。
- 注册成功后默认自动登录，并进入百度网盘授权引导。

**异常提示**：

- 账号格式错误。
- 密码不符合要求。
- 两次密码输入不一致。
- 验证码错误或过期。
- 账号已存在。
- 网络异常，请稍后重试。

### 8.2 账号登录与退出

**目标**：让已有用户进入个人备份空间，并支持退出当前账号。

**规则**：

- 支持手机号或邮箱登录。
- 支持记住登录状态。
- 支持忘记密码入口。
- 连续登录错误时，需要安全验证或短暂限制。
- 用户退出登录后，不影响已完成备份记录。
- 用户重新登录后，可继续查看历史任务和异常中断任务。

**异常提示**：

- 账号不存在。
- 密码错误。
- 账号状态异常。
- 网络异常，请稍后重试。

### 8.3 百度网盘授权

**目标**：让用户授权产品将备份文件保存到自己的百度网盘。

**授权状态**：

- 未授权。
- 已授权。
- 授权已失效。
- 授权异常。

**用户可见信息**：

- 百度网盘账号昵称。
- 授权状态。
- 最近授权时间。
- 重新授权入口。
- 解除绑定入口。
- 网盘空间概况。

**规则**：

- 未授权时，用户不能开始备份任务。
- 授权失效后，用户不能继续备份，必须重新授权。
- 用户可主动解除绑定。
- 解除绑定前必须二次确认，并提示会影响正在进行或后续备份任务。
- 用户主动解除绑定后，未完成任务不能继续，直到重新授权。

### 8.4 设备绑定

**目标**：将当前设备纳入账号管理，用于区分任务来源与历史备份记录。

**设备信息**：

- 设备名称。
- 设备类型。
- 当前设备标识。
- 首次绑定时间。
- 最近在线时间。
- 最近备份时间。
- 备份任务数量。
- 备份状态概况。

**规则**：

- 新设备登录后，系统提示绑定当前设备。
- 系统提供默认设备名称，用户可修改。
- 同一账号可绑定多台设备。
- 当前设备未绑定时，用户可查看账号信息，但不能创建和执行备份任务。
- 设备名称允许重复，但应建议用户使用易识别名称。

### 8.5 设备管理

**目标**：让用户查看和管理账号下所有已绑定或曾绑定设备。

**设备状态**：

- 当前在线。
- 最近在线。
- 长时间未使用。
- 已解绑。

**支持操作**：

- 查看设备列表。
- 查看当前设备标识。
- 修改设备名称。
- 查看设备备份概况。
- 查看某台设备下的备份任务。
- 解绑设备。

**解绑规则**：

- 用户可解绑非当前设备。
- 用户解绑当前设备前必须二次确认。
- 解绑设备不删除百度网盘中已完成的备份文件。
- 解绑设备不影响历史备份记录查看。
- 解绑后，该设备再次使用时需要重新绑定。

### 8.6 创建备份任务

**目标**：让用户选择文件或文件夹，并配置基础备份选项。

**可选内容**：

- 单个文件。
- 多个文件。
- 单个文件夹。
- 多个文件夹。
- 文件与文件夹混合选择。

**任务设置项**：

| 设置项 | 默认值 | 说明 |
| --- | --- | --- |
| 任务名称 | 自动生成 | 用户可修改 |
| 备份内容 | 用户选择 | 文件或文件夹 |
| 加密备份 | 开启 | 用户可关闭，关闭时需确认风险 |
| 去重备份 | 开启 | 首期默认开启，不建议关闭 |
| 任务备注 | 空 | 可选 |

**创建前校验**：

- 未登录时，不允许进入创建流程。
- 百度网盘未授权时，不允许创建备份任务。
- 当前设备未绑定时，不允许创建备份任务。
- 未选择文件或文件夹时，不允许继续。
- 所选项目不存在或无权限访问时，提示用户处理。
- 百度网盘空间不足时，提示用户清理空间或调整备份内容。

**任务创建结果**：

- 用户确认去重分析结果后，任务进入任务列表。
- 新任务初始状态为“待开始”。
- 用户需要点击“开始”后才进入实际备份流程。

### 8.7 打包与加密

**目标**：将用户选择的内容整理为适合保存到百度网盘的备份包，并默认加密。

**规则**：

- 备份任务默认开启加密。
- 用户可在创建任务时关闭加密。
- 关闭加密前必须展示风险确认。
- 任务开始后，加密设置不可随意变更。
- 加密任务完成后，在任务和记录中展示“已加密”标识。
- 未加密任务完成后，在任务和记录中展示“未加密”标识。

**关闭加密确认文案**：

关闭加密后，备份内容将以未加密形式保存。请确认该任务不包含敏感资料。

### 8.8 多设备去重分析

**目标**：备份前对比当前账号下历史备份项目，跳过重复内容。

**去重范围**：

- 当前账号下已绑定设备的历史备份项目。
- 当前账号下曾绑定设备的历史备份项目。
- 当前设备内已备份过的项目。

**去重结果分类**：

| 类型 | 说明 |
| --- | --- |
| 需备份 | 本次需要实际备份的项目 |
| 已备份 | 已在当前账号的其他任务中备份过 |
| 重复跳过 | 本次不再重复备份的项目 |
| 异常待确认 | 当前无法确认状态，需要用户处理或稍后重试 |

**展示要求**：

- 展示本次选择总项目数。
- 展示预计需要备份的项目数。
- 展示可跳过的重复项目数。
- 展示异常待确认项目数。
- 展示预计节省空间。
- 支持展开查看重复项目详情。
- 支持按设备来源查看重复项目来自哪台设备。
- 支持查看重复项目来自哪个历史任务。

**用户理解要求**：

- 页面应说明“这些内容已经在你的其他设备或历史任务中备份过，本次会自动跳过”。
- 不要求用户理解文件指纹、哈希、分块等技术概念。

### 8.9 备份任务管理

**目标**：让用户在任务列表中查看、操作和管理所有备份任务。

**任务状态**：

| 状态 | 说明 |
| --- | --- |
| 待开始 | 任务已创建，但尚未开始 |
| 准备中 | 正在整理备份内容 |
| 对比去重中 | 正在生成去重结果 |
| 备份中 | 正在备份 |
| 已暂停 | 用户主动暂停 |
| 恢复中 | 正在从暂停或异常中断状态恢复 |
| 异常中断 | 因断网、关闭应用、设备关机等原因中断 |
| 备份失败 | 任务无法继续，需要用户处理 |
| 已完成 | 所有需备份项目已完成 |
| 已删除 | 用户已删除任务记录 |

**支持操作**：

- 开始任务。
- 暂停任务。
- 恢复任务。
- 异常中断后继续任务。
- 删除任务。
- 查看任务详情。
- 查看备份进度。
- 查看备份结果。

**删除规则**：

- 待开始、已暂停、异常中断、备份失败、已完成的任务可删除。
- 正在备份的任务需要先暂停或二次确认后删除。
- 删除任务记录不默认删除百度网盘中已完成的备份文件。
- 删除前必须明确提示删除影响。

**删除确认文案**：

删除后，该任务将不再显示在任务列表中。已完成上传到百度网盘的备份文件不会自动删除。

### 8.10 备份进度

**目标**：让用户理解任务当前处理到哪一步、已经完成什么、接下来还会继续什么。

**进度信息**：

- 当前任务状态。
- 总项目数。
- 已完成项目数。
- 跳过重复项目数。
- 异常项目数。
- 当前正在处理的项目名称。
- 已用时间。
- 预计剩余时间。
- 当前备份速度。
- 备份目标位置。

**展示规则**：

- 任务列表展示简洁进度条和状态。
- 任务详情展示完整分项进度。
- 异常信息需要说明发生了什么、可能原因和下一步建议。
- 完成后展示备份结果汇总。

### 8.11 异常处理与恢复

**目标**：任务异常后帮助用户理解问题，并尽可能继续任务。

**异常类型与处理**：

| 异常类型 | 用户提示 | 用户操作 |
| --- | --- | --- |
| 网络异常 | 网络连接异常，备份已暂停 | 检查网络后继续 |
| 应用关闭 | 上次任务未正常完成 | 继续备份 |
| 设备关机 | 任务因设备离线中断 | 重新打开后继续 |
| 授权失效 | 百度网盘授权已失效 | 重新授权 |
| 空间不足 | 百度网盘空间不足 | 清理空间后继续 |
| 文件不可访问 | 部分文件无法读取 | 跳过或重新选择 |
| 用户取消 | 用户已取消任务 | 重新开始或删除 |

**恢复规则**：

- 异常中断任务必须保留在任务列表中。
- 用户重新进入产品后，应看到明确恢复入口。
- 已完成项目不重复备份。
- 无法恢复的任务，应提示原因和下一步建议。
- 恢复前应重新检查授权状态、设备状态、网络状态、网盘空间和文件可访问状态。

### 8.12 备份记录

**目标**：让用户查询历史备份任务与结果。

**记录字段**：

- 任务名称。
- 所属设备。
- 创建时间。
- 完成时间。
- 备份内容概况。
- 是否加密。
- 去重结果。
- 任务状态。
- 百度网盘保存位置。

**查询筛选**：

- 按设备筛选。
- 按任务状态筛选。
- 按时间筛选。
- 按是否加密筛选。
- 按任务名称关键词搜索。

## 9. 页面规格

### 9.1 启动页

- 展示产品名称或品牌标识。
- 展示加载状态。
- 已登录用户进入首页。
- 未登录用户进入登录页。
- 登录状态异常时，引导重新登录。

### 9.2 注册页

- 提供手机号或邮箱、验证码、密码、确认密码输入。
- 提供用户协议与隐私政策勾选。
- 提供注册按钮和登录入口。
- 注册成功后进入授权引导。

### 9.3 登录页

- 提供账号、密码输入。
- 提供记住登录状态、忘记密码、注册入口。
- 登录成功后进入首页或继续未完成引导。
- 登录失败展示明确原因。

### 9.4 授权引导页

- 说明为什么需要授权百度网盘。
- 提供授权百度网盘按钮。
- 展示授权状态。
- 授权失败时提供重试入口。
- 首期不允许跳过授权后创建备份任务。

### 9.5 设备绑定页

- 展示默认设备名称和设备类型。
- 允许用户修改设备名称。
- 提供绑定按钮。
- 绑定成功后进入首页。
- 绑定失败时提供重试提示。

### 9.6 首页 / 工作台

- 展示账号信息。
- 展示百度网盘授权状态。
- 展示当前设备状态。
- 展示备份任务概览。
- 展示最近任务列表。
- 展示创建备份任务入口。
- 展示异常任务提醒。
- 展示百度网盘空间提醒。

### 9.7 创建备份任务页

- 支持输入任务名称。
- 支持选择文件和选择文件夹。
- 展示已选择项目列表。
- 支持移除和继续添加项目。
- 展示加密备份开关，默认开启。
- 关闭加密时弹出风险确认。
- 支持填写任务备注。
- 点击下一步进入去重分析页。

### 9.8 去重分析页

- 展示分析中状态。
- 展示总项目数、需备份数量、重复跳过数量、异常待确认数量和预计节省空间。
- 支持展开重复项目详情。
- 支持按设备和历史任务查看重复来源。
- 异常项目支持跳过或返回处理。
- 用户确认后创建任务。

### 9.9 备份任务列表页

- 提供任务搜索。
- 提供状态、设备、时间筛选。
- 展示任务名称、所属设备、创建时间、当前状态、进度、是否加密、去重摘要和操作按钮。
- 不同任务状态展示不同操作。
- 异常中断任务突出展示继续入口。
- 删除任务前二次确认。

### 9.10 备份任务详情页

- 展示任务基础信息、当前状态、总体进度、分项进度、当前处理项目、去重结果、备份内容、异常项目、操作区和结果汇总。
- 完成后展示实际备份项目数、重复跳过项目数、异常项目数、节省空间和百度网盘保存位置。

### 9.11 设备管理页

- 展示设备列表。
- 明确标记当前设备。
- 展示设备状态和备份概况。
- 支持重命名设备。
- 支持查看设备任务。
- 支持解绑设备，解绑前二次确认。

### 9.12 百度网盘管理页

- 展示授权状态。
- 展示百度网盘账号信息。
- 展示最近授权时间。
- 展示网盘空间概况。
- 提供重新授权按钮。
- 提供解除绑定按钮，解除前提示影响。

### 9.13 备份记录页

- 提供搜索、时间筛选、设备筛选、状态筛选、加密筛选。
- 展示历史记录列表。
- 点击记录进入任务详情。
- 支持查看已完成、失败和删除前记录。

### 9.14 设置页

- 展示账号信息。
- 提供修改密码入口。
- 提供默认加密设置。
- 提供百度网盘授权入口。
- 提供设备管理入口。
- 提供退出登录。
- 关闭默认加密时必须风险确认。

### 9.15 异常与通知中心

- 展示备份完成、任务暂停、任务异常中断、授权失效、空间不足、文件不可访问、设备绑定异常等通知。
- 点击通知进入对应任务或设置页。
- 已处理通知自动标记。
- 用户可手动清除通知。

## 10. 关键状态

### 10.1 空状态

- 首页无任务：提示用户创建第一个备份任务。
- 设备列表为空：提示用户先绑定当前设备。
- 备份记录为空：提示完成备份后会显示记录。

### 10.2 加载状态

- 正在登录。
- 正在授权。
- 正在绑定设备。
- 正在读取文件信息。
- 正在分析重复项目。
- 正在创建任务。
- 正在备份。
- 正在恢复任务。

### 10.3 错误状态

- 网络连接失败。
- 授权失败。
- 设备绑定失败。
- 文件不可访问。
- 百度网盘空间不足。
- 任务恢复失败。
- 删除任务失败。

错误提示必须包含：

- 发生了什么。
- 为什么可能发生。
- 用户下一步可以做什么。
- 重试入口。

## 11. 验收标准

### 11.1 账号与登录

- 用户可以完成注册、登录和退出登录。
- 未登录用户不能进入需要账号的功能页面。
- 登录异常时有明确提示。
- 注册成功后可进入首次使用引导。

### 11.2 百度网盘授权

- 未授权用户能看到授权引导。
- 授权成功后能看到授权状态和账号信息。
- 授权失效后能看到重新授权入口。
- 解除绑定前有二次确认。
- 未授权或授权失效时，用户不能开始备份。

### 11.3 设备管理

- 用户可以绑定当前设备。
- 用户可以查看设备列表。
- 当前设备有明确标识。
- 用户可以重命名设备。
- 用户可以解绑设备。
- 解绑设备前有二次确认。
- 解绑设备不影响历史备份记录查看。

### 11.4 备份任务

- 用户可以选择文件或文件夹创建备份任务。
- 默认加密开启。
- 用户关闭加密时有风险确认。
- 创建任务前能看到去重分析结果。
- 任务可以开始、暂停、恢复、删除。
- 任务异常中断后可以继续。
- 任务详情页能展示完整进度和结果。
- 删除任务记录不会默认删除百度网盘备份文件。

### 11.5 多设备去重

- 系统能展示本次需备份项目和重复跳过项目。
- 用户能查看重复项目来自哪台设备或历史任务。
- 完成备份后，结果中展示实际备份数量、跳过数量和节省空间。
- 去重说明不暴露复杂技术细节。

### 11.6 页面体验

- 首页能展示任务、设备、授权的核心状态。
- 各页面空状态、加载状态、错误状态完整。
- 高影响操作均有二次确认。
- 异常信息有明确下一步操作建议。

## 12. 待确认问题

以下问题来自 PRD，仍建议在进入原型和研发计划前确认：

1. 首期是否严格只做桌面端，还是同时提供网页管理端。
2. 注册方式是手机号优先、邮箱优先，还是两者都支持。
3. 是否允许一个设备切换绑定不同账号。
4. 备份完成后是否需要提供“恢复到本地”的功能。
5. 删除任务时，是否需要提供同时删除百度网盘备份文件的选项。
6. 是否需要设置单个账号可绑定设备数量上限。
7. 是否需要支持定时自动备份。
